# Anomaly Detection Using Artificial Intelligence for Network Security

This project implements a machine learning-based anomaly detection system using Python.

## Features
- Classifies network traffic as normal or abnormal
- Uses Logistic Regression model
- Detects anomalies and generates alerts
- Evaluates performance using accuracy

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Results
- Achieved 100% accuracy on sample dataset
- Successfully detected abnormal traffic patterns
- 
## How It Works
The model is trained on labeled network traffic data and tested on unseen data to detect abnormal patterns.
